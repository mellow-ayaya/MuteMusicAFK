### MuteMusicAFK

Automatically mutes music when AFK logout starts to disable music in the character select and login menu.
### &nbsp;
### About

This addon has no UI and no settings. It will mute your music in the last second of the automatic AFK logff to mute the super dramatic, continously repeating character select and login menu music.

When automatically muting music on AFK logoff, the addon will also automatically re-enable the music on the next login.
